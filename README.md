# Task 4 - Basic Weather App

## Objective
To create a Python program that fetches the current weather of any city using OpenWeatherMap API.

## Tools Used
- Python
- requests library
- OpenWeatherMap API
- Google Colab (optional)

## Steps Performed
1. User inputs the city name.
2. Program sends request to OpenWeatherMap API.
3. Response is parsed to get temperature and weather description.
4. Results are displayed to the user.

## Outcome
The program successfully provides real-time weather information for the specified city.
